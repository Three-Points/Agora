### Getting Started

#### Installation

For start to develop components into this project follow the steps below:

1. Install the dependencies (Use **npm** preferably):

    ```bash
    npm install
    ```

2. Run the application in mode `storybook`:

    ```bash
    npm run storybook
    ```
